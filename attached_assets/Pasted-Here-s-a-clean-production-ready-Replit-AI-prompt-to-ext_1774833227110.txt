## ⚙️ Backend Logic

Update quote calculation function:

```ts
function calculateQuote(data) {
  const residentialPrice = baseCalculation(data);

  if (data.moveType === "commercial") {
    const doubled = residentialPrice * 2;
    const tierMinimum = getCommercialTierMinimum(data);

    return Math.max(doubled, tierMinimum);
  }

  return residentialPrice;
}
```
