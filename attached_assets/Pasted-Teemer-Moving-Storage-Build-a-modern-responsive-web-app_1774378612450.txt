Teemer Moving Storage

Build a modern, responsive web application for a moving company called “Teemer Moving & Storage” with two main product experiences:

1. Information Website
2. Teemer Moving Platform

The splash screen / landing page should act as the front door to both experiences and give users a clear choice between:
- Enter Information Site
- Request / Find a Moving Job

The product should feel like a premium, tech-enabled logistics platform — a real “Uber for Moving” experience — while still preserving trust, professionalism, and the real-world moving company feel.

TECH STACK
Use:
- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript
- Reusable component architecture
- Clean file structure
- Mobile-first responsive design

- Stripe
- Mapbox
- Realtime tracking via websockets
